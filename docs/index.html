MSFTBandApp

A third-party Microsoft Band client application. Created by ilmiont (James Walker).

  • GitHub
  • MSFTBandLib (third-party Microsoft Band SDK powering this app)

MSFTBandApp is a third-party Microsoft Band app developed to allow Microsoft Band owners to continue using their devices beyond the official end of support in May 2019.

This app is currently in very early development and is highly experimental. Please see the news feed below for details on the status of the project.

The long-term aim is to achieve feature parity with the official Microsoft Band app, prior to its demise, with builds available for UWP and Android and hardware support for Microsoft Band 1 (Cargo) and Microsoft Band 2 (Envoy).

This app is developed using the MSFTBandLib third-party Microsoft Band SDK implementation, which is also created by this author.

MSFTBandApp and MSFTBandLib are open-source projects licensed under the MIT License.
